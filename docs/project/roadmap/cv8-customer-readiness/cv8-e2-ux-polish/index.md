# CV8.E2 — UX Polish

**CV:** [CV8 — Customer Readiness](../index.md)
**Status:** ⚪ Planned

---

## What This Is

The set of small things that distinguish a working product from a polished one. Empty states designed (never blank), loading skeletons (never spinners), error states with retry, saved views, and the ⌘K command palette.

---

## Stories

| Code | Story | Status |

| CV8.E2.S3 | Error states with retry / contact action | 📥 Backlog |
| CV8.E2.S4 | Saved views & favorites surface | 📥 Backlog |
| CV8.E2.S5 | ⌘K command palette (jump to page, run common action) | 📥 Backlog |
| CV8.E2.S6 | Toasts and inline messaging vocabulary defined | 📥 Backlog |

---

## Done Condition

- No page shows raw white space when empty.
- No page shows a generic spinner; every async surface has a skeleton.
- The cmd palette covers the 10 most common jumps.
