# CV0.E3 — Assessment Report Template

**CV:** [CV0 — Pilot Data-Quality Assessment](../index.md)
**Status:** ⚪ Planned
**Depends on:** CV0.E1, CV0.E2

---

## What This Is

The report packages the methodology + tooling output for a non-engineer audience: prospect executives, maintenance directors, procurement leaders. It is the artifact a customer can take to their CFO to justify a pilot.

The report distinguishes what SmartInv can deliver today (confidence band X) from what requires cleansing first (uplift to confidence band Y) — explicitly, in dollar and risk terms.
